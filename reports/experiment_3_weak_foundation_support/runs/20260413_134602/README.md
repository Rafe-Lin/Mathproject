# Experiment 3: Weak Foundation Support (Escape from C)

## Research Question
RQ3: Can Adaptive support help weak students escape the C range?

## Fixed Setting
- Student group: Weak only（減C組）
- Success threshold: 0.60
- Goal: escape from C, not reach A

## Core Findings
- Adaptive support improves weak students’ final mastery.
- Escape-from-C rate increases with support.
- Gains eventually show diminishing returns.
- Experiment 3 identifies a practical support range, not an unlimited-support optimum.
