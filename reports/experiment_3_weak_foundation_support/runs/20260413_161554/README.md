# Experiment 3: Weak Escape-from-C under Adaptive (Policy A)

## Research Question
Can Adaptive help weak students escape from C to B (mastery >= 0.60)?

## Method
- Weak only
- AB3 only
- Total-step relaxation only (no forced intervention)
- MAX_STEPS sweep: 30, 40, 50, 60, 70

## Key Findings
1. Escape-from-C rate changes with total-step budget.
2. Additional steps improve outcomes but may show diminishing returns.
3. Cost-effectiveness is reported via cost_per_1pct_escape.
